收藏与发掘价值，从被动记录到主动洞察

核心理念

策展人

Curator

不只是收集，而是在收藏与策展。通过独特视角，发掘最具价值的条目。

情报

Intelligence

从被动记录转向主动挖掘。洞察偏好，发现数据背后的关联网络。

绝对隐私

Privacy First

本地优先架构，数据完全私有。具备隐私保护与伪装能力。

东方美学

CJK Aesthetics

专为东方文字设计的竖排 UI，还原阅读实体杂志的沉浸感。

零摩擦

Zero-Friction

极致的自动化采集与整理，让繁琐的录入过程几乎不可感知。

功能亮点

沉浸式采集

浏览器插件与本地服务联动，浏览即收藏。收件箱工作流自动完成元数据抓取与清洗，确认归档即可。

智能管理

拟物化收藏体验、ForceView 关系图展示关联网络、画廊模式、时间轴与 Tag 智能筛选。

量化自我

行为记录与偏好分析，通过数据洞察变化趋势，生成可定制的分析报告。

极致安全

一键伪装、本地化存储。数据与资源均存于本地，不上传云端。

技术架构

UI PySide6 (Qt)
Backend FastAPI
Database SQLite + SQLAlchemy
Crawler 模块化多源抓取

单体核心 + 可选插件架构，本地优先，无云依赖。
