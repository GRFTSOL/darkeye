去fanza上会看到每部片子的信息包括


| 日文    | 英文     | 中文   | 解释                             |
| ----- | ------ | ---- | ------------------------------ |
| 配信開始日 |        |配信开始日| 网络发布日，一般来说网络早于配信，网络发布后就有盗版     |
| 商品発売日 |        |商品发售日| 线下作品DVD发布日，有了后就可以在实体店买到了                    |
| 収録時間  |        |收录时间 |片子时间                       |
| 出演者   |        |出演人员|一般只写女演员，不会写男演员             |
| 監督    |        |导演 |一般一部片子只有一个，偶尔会有多个，还有很多不写，那是因为成熟的骗片子似乎不写                          |
| シリーズ  | series | 系列   |                                |
| メーカー  | maker  | 制作商  |                                |
| レーベル  | label  | 厂牌   |                                |
| ジャンル  | Genre  | 类型   |                                |
| 品番    |        | 产品编号 | 例如IPX-247但是fanza内部有一个特别的编号|



AV没有艺术性，或者说是极端商业导向的，一年可能都没有10部作品能有艺术性，能把故事讲好。全是高度工业流水线，一天拍摄结束的东西。

AV拍摄流程


fanza上面一部片最多有5项
包括

| 序号 | 类型           | 说明|
| --- | --- | --- |
| 1   | 动画           |这个好像是一次性的在网上看的|
| 2   | dvd通贩        |这个就是实体|
| 3   | blu-ray通贩    |这个是实体的蓝光盘|
| 4   | 月额动画普通    |说明这个片包括在包月里面|
| 5   | 月额动画4K      |说明这个片包括在包月4K里面|

有的普通的片在fanza上卖，就是没有实体的，而且价格也便宜。

像sod prestige 等厂商是不在fanza卖实体的，但是有动画卖。比如小凑四叶的[[SDHS-057]],这种就是连dvd封面都没有。


sod的片在fanza上没有标准的大图dvd封面，只有半封面。

prestige在fanza上也没有大图封面。
