# 软件定位


PC端个人记录jav数据并统计，创新推荐方式与搜索方式，为了挖掘最艺术的作品，更好的作品之间的联系。

软件劣势，在挖掘新的片子上存在劣势。

黄网的优势-ugc打分系统，ugc评论系统，最新的新人推荐，热点推荐，更加全面的搜索，除了jav，其他各种都有，信息库全。

刮削类的项目的优势，但是后续的数据清洗的，收藏的，管理的分析的都非常的少，nfo数据无法处理多语言的tag和演员多马甲问题。很多刮削类项目爬一堆没有价值的中文网站。

Jvideo/jellyfin/Kodi/Emby 的优势标准化，够用，但是不够专注，没有特殊的功能。
