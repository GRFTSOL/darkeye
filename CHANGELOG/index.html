    ## 1.x.x(路线图)

    • [ ] C++重写稳定模块
    • [ ] tag计算器，高级tag过滤系统，tag重定向
    • [ ] 重构tag系统，弄成三层tag匹配i18n,加上良好的同步方式
    • [ ] graph系统的更改，关联算法，社区发现算法，过滤，把note互联后的计算上
    • [ ] 重绘图,用qchart移除对matplotlib的依赖
    • [ ] 作品信息文本导出
    • [ ] 刮削保存到原视频文件夹里面
    • [ ] 爬虫数据流节点化，就像ue5蓝图那样

    • [ ] 爬虫的重构与插件化爬虫

    • [ ] 详情页播放按钮放大-这个需求应该没有了，有的页面已经不用了

    • [ ] 刮削保存到原视频文件夹里面,这个等后面爬虫功能完善了
    • [ ] FC2爬取支持

    • [ ] 冷门番号爬取支持

    • [ ] dmm 1080p分辨率封面下载

    • [ ] 科普知识与serverless更新

    • [ ] 绿色模式，现在实现了一半，不过貌似没用

    • [ ] i18n,未来打算支持简体中文，繁体中文，日文，韩文与一切能支持竖排的语言。
    • [ ] 以竖排风格为核心，优化UI的界面

    • [ ] 编辑界面高级停靠系统的保存，包括高级停靠区的继续开发

    • [ ] Jvideo/jellyfin/Kodi/Emby的数据导入，这个现在变成统一的nfc数据的导出

    • [ ] 其他格式的信息导出功能，目前只能导.csv，还没想好怎么导出

    • [ ] NFO数据的导出，挂削数据的导出

    • [ ] 全部后台线程读写数据库

    • [ ] 提示封面是否高清
    • [ ] notes互联，通过[[movie:xxxx]] [[actress:xxxx]] [[actor:xxxx]]中种方式实现互联与跳转，包括预览，加入图系统
    • [ ] 更好的翻译，现在默认的google翻译不太好用。然后爬取的东西也不好，先考虑技术方案，接入本地大模型

    • [ ] graph加上series的互联

    • [ ] 附加更多的人工标注信息包括女优的状态，等等。时长进度条标注东西。

    • [ ] 新版编译后至少50个爬虫的测试，包括新爬测试，里面包含的番号要全，正常的就不说了，但是冷门的要包含
    • [ ] dvd展开的重构，包括可以看背面，dvd不同宽度的盒子，更多的动画可以看dvd背面
    • [ ] HXAD-015片商爬虫问题
    • [ ] NFO导入优化问题，男女优判断问题，女优多马甲问题，无番号问题。
    • [ ] 随机展示
    • [ ] 显示身材类型
    • [ ] NFO导入时男女优的区分，多名字的区分与合并
    • [ ] 接入mdcx的nfo导入
    • [ ] 视频文件名缓存
    • [ ] fanza大图下载与大图提示
    • [ ] NFO数据的导出

    • [ ] 过滤栏不够宽滚动与combobox滚动冲突问题

    • [ ] 恢复公库与私库时校对数据库版本，这个看看有没有必要

    • [ ] 迁移爬虫去浏览器插件，保留设计标准接口，软件仅保留图片下载的功能。
    • [ ] 项目整理，代码的结构调整
